# Teams App Icons

Replace these placeholder files with proper icons before sideloading:

- `color.png` — 192x192 full-color app icon
- `outline.png` — 32x32 transparent outline icon (white on transparent)

## Sideloading

1. Replace the icon files
2. Zip manifest.json + color.png + outline.png into a .zip file
3. In Teams Admin Center or Teams client, upload the .zip as a custom app
